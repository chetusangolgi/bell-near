# NEAR Blockchain Notification System

## Introduction

The NEAR Blockchain Notification System is designed to provide users with real-time notifications regarding various events and transactions happening on the NEAR blockchain. This system aims to enhance user experience by ensuring timely updates and alerts, allowing users to take necessary actions promptly.

## Features
- **Real-time Notifications**: Get instant alerts for transactions, changes in account status, smart contract events, and more.
- **Configurable Alert Settings**: Users can customize their notification preferences based on specific events and triggers.
- **User-friendly Interface**: An intuitive dashboard that allows users to view their notifications and manage settings easily.

## Getting Started

To set up the NEAR Blockchain Notification System, follow these steps:
1. Clone the repository: `git clone https://github.com/chetusangolgi/bell-near.git`
2. Install the necessary dependencies.
3. Configure your NEAR account and set the desired notification preferences.

## Usage

Once set up, the system will automatically start monitoring the NEAR blockchain for events. Notifications will be sent based on your configured settings.
